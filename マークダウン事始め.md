# 組合の取り込み方

## 取り込み方3つ
- 純額法
- 中間法
  - 折衷法
  - Gross-Net法
- 総額法

### 総額法について
1. 期末に純資産以外の科目を取り込み、出資金勘定と相殺
2. 翌期首にBS科目だけ洗い替え
3. その結果、出資金勘定は純額法と同じ金額になる

| 組合 | 任意組合系 | 匿名組合 |
| :-: | :-: | :-: |
| 配当時の源泉税 | 控除しない | 控除する |
